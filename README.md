## Coffee Shop Fullstack Web Application

## Description
Fullstack web app using Spring Boot and ReactJS for managing coffee products, cart and orders.

## Tech Stack
- Backend: Spring Boot
- Frontend: ReactJS
- Database: MySQL

## Run
Backend:
mvn spring-boot:run

Frontend:
cd react-shop
npm install
npm start

## Features
- Login (JWT)
- Product CRUD
- Cart & Order
